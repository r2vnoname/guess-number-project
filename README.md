 guess-number-project
Игра "Угадай число"
Угадай число с ИИ-советником

Описание проекта
Проект по информатике, демонстрирующий работу алгоритмов обработки данных на основе базы данных SQLite. Программа реализует классическую игру "Угадай число", но с интеллектуальной аналитической надстройкой. Все попытки всех игр сохраняются в базу данных, после чего алгоритм сравнивает эффективность игрока с историческими данными.

Используемые технологии
- Язык: Python 3.x
- База данных: SQLite (встроенная библиотека `sqlite3`)
- Интерфейс: Консольное меню

Реализованные алгоритмы

1. Алгоритм бинарного поиска с обратной связью
В процессе игры программа генерирует подсказки "больше" / "меньше", реализуя классический алгоритм бинарного поиска. Игрок должен следовать подсказкам для оптимального угадывания.

2. Алгоритм накопления данных
Каждая попытка каждого игрока сохраняется в таблицу `attempts` базы данных SQLite. Накапливается история всех игр с полной информацией: загаданное число, догадка, подсказка, номер попытки.

3. Алгоритм анализа эффективности (ключевой)
После завершения игры программа выполняет аналитический запрос к базе данных:
- Находит все прошлые игры с таким же загаданным числом.
- Вычисляет среднее арифметическое количества попыток.
- Находит минимальное количество попыток (лучший результат).
- Сравнивает результат текущего игрока со средним значением и выдаёт персонализированное заключение ("лучше среднего", "хуже среднего", "на уровне").

4. Алгоритм ранжирования (ТОП-5)
Дополнительный алгоритм, который запрашивает из базы данных 5 игр с наименьшим количеством попыток, сортируя их по возрастанию.

Структура базы данных

Таблица `attempts`:
Поле            Тип       Описание
id              INTEGER   Уникальный идентификатор попытки
game_id         INTEGER   Номер игры
secret_number   INTEGER   Загаданное число (1–100)
guess           INTEGER   Число, введённое игроком
hint            TEXT      Подсказка: 'higher', 'lower', 'correct'
attempt_number  INTEGER   Номер попытки внутри игры
